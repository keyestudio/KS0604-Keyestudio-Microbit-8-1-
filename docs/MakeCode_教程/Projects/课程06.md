### 3.2.6 智能追光狗

#### 3.2.6.1 简介

![Img](./media/top1.png)

使用一些乐高积木块、Microbit V2主板，舵机扩展板和乐高舵机等，来搭建一个智能追光狗，通过代码编程来控制智能追光狗追光运动。

![Img](./media/bottom1.png)

#### 3.2.6.2 元件知识

![Img](./media/2top.png)

**Microbit光线接收**

![Img](./media/j101.png)

micro:bit主板并不自带光敏传感器，它对外界光照强度的检测是通过5×5LED点阵屏进行的，是利用5×5LED点阵屏作为光接收元件。5×5LED点阵屏被用来感知周围的光，并反复地将光照强度通过5×5LED点阵屏转换成输入，并采样电压衰减时间，这样检测出来的光照强度是一个相对值。（**注意：** 将光线亮度输出至串口，输出的是一个相对值。）

#### 3.2.6.5 代码流程图

```mermaid
flowchart TD
    Start["开始"] --> OnInit1["初始化"]

    OnInit1 --> OnInit2["设置SK6812 RGB LED的引脚为P0<br>设置SK6812 RGB LED亮度为30%<br>SK6812 RGB LED不亮"]
    
    OnInit2 --> Forever["进入主循环 (forever)"]
    
    Forever --> Conditionals{检测光线等级}

    Conditionals -- 光线等级 ≥ 200 --> Conditionals1[高亮条件]
    
    Conditionals1 --> EventA["1\. SK6812 RGB LED亮红灯<br>2\. 5×5LED点阵屏显示“高兴”图案<br>3\. 智能追光狗后退"]
    EventA --> Conditionals

    Conditionals -- 10 < 光线等级 < 200 --> Conditionals2[中等亮度条件]

    Conditionals2 --> EventB["1\. SK6812 RGB LED亮蓝灯<br>2\. 5×5LED点阵屏显示“困惑”图案<br>3\. 智能追光狗前进"]
    EventB --> Conditionals

    Conditionals -- 光线等级 ≤ 10 --> Conditionals3[低亮/黑暗条件]

    Conditionals3 --> EventC["1\. SK6812 RGB LED熄灭<br>2\. 5×5LED点阵屏清屏<br>3\. 智能追光狗停止"]
    EventC --> Conditionals
```

#### 3.2.6.7 实验结果

![Img](./media/4top.png)

按照接线图接好线，利用micro USB数据线上电，同时还需要外接电源 (4个AAA电池安装到电池盒，且保证电源充足)。

使用在线浏览器下载示例代码，则需要将下载好的 “.hex” 文件发送到micro:bit主板。

**① 下载示例代码(WebUSB功能)**

使用 **Google Chrome** 浏览器 或  **Microsoft Edge** 浏览器，将示例代码下载到micro:bit主板上：

![Img](./media/Animation-06.gif)

**② 下载示例代码(非WebUSB功能)**

使用其他浏览器（非Google Chrome 或 Microsoft Edge），将示例代码下载、发送到micro:bit主板上：

![Img](./media/Animation-006.gif)

将示例代码成功下载、发送到micro:bit主板之后，然后将micro USB数据线从micro:bit主板上拔下来。

移动光源（或者打开手机照明）照射到micro:bit上面的5×5LED点阵屏。

当光线等级 ≥ 150时，智能追光狗随光源后退，同时SK6812 RGB LED亮红灯，micro:bit上面的5×5LED点阵屏显示图案![Img](./media/ab2.png)；

当 10 < 光线等级 < 150时，智能追光狗随光源前进，同时SK6812 RGB LED亮蓝灯，micro:bit上面的5×5LED点阵屏显示图案![Img](./media/ab3.png)；

当光线等级 ≥ 10时，智能追光狗停止，同时SK6812 RGB LED熄灭，micro:bit上面的5×5LED点阵屏清屏。

![Img](./media/dongtu-6.gif)
