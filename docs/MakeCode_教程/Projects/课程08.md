### 3.2.8 雄鹰翱翔

#### 3.2.8.1 简介

![Img](./media/top1.png)

使用一些乐高积木块、Microbit V2主板，舵机扩展板和1个360°乐高舵机等，来搭建一个雄鹰，通过Microbit V2主板上面的按钮A/B和代码编程来控制雄鹰翱翔。

#### 3.2.8.4 代码流程图

```mermaid
flowchart TD
    Start([开始]) --> OnInit1[初始化]
    OnInit1 --> OnInit2[设置SK6812 RGB LED的引脚为P0<br>设置SK6812 RGB LED亮度为30%<br>SK6812 RGB LED不亮]
    OnInit2 --> IdleState[空闲状态/等待事件]
    IdleState --> EventBtnA{按下micro:bit V2主板上面的按钮A且松开 ?}
    IdleState --> EventBtnB{按下micro:bit V2主板上面的按钮B且松开 ?}
    IdleState --> EventBtnAB{同时按下micro:bit V2主板上面的按钮A+B且松开 ?}
    EventBtnA -->|否| IdleState
    EventBtnA -->|是| BtnAAction[1\. SK6812 RGB LED亮红灯<br>2\. 5×5 LED点阵屏显示“高兴”图案<br>3\. 雄鹰缓慢飞舞起来]
    BtnAAction --> IdleState
    EventBtnB -->|否| IdleState
    EventBtnB -->|是| BtnBAction[1\. SK6812 RGB LED亮绿色灯<br>2\. 5×5 LED点阵屏显示“奶牛”图案<br>3\. 雄鹰快速飞舞起来]
    BtnBAction --> IdleState
    EventBtnAB -->|否| IdleState
    EventBtnAB -->|是| BtnABAction[1\. SK6812 RGB LED熄灭<br>2\. 5×5 LED点阵屏清屏<br>3\. 雄鹰停止飞舞]
    BtnABAction --> IdleState
```

#### 3.2.8.5 指令方块说明

![Img](./media/55LED.png)

![Img](./media/RGBLED.png)

![Img](./media/RGBLED1.png)

![Img](./media/RGBclear.png)

6\. 当按下（或同时按下）micro:bit V2主板上面的按钮（A、B 或A+B）并再次松开时，触发并执行代码块中的一系列动作。

![Img](./media/buttonAB.png)

7\. 设置SK6812 RGB LED点亮的颜色灯.

![Img](./media/color.png)

8\. 设置360°舵机的引脚，转动方向(顺时针/逆时针)和转动的速度(0~100%)。

![Img](./media/360Servo.png)

#### 3.2.8.7 实验结果

![Img](./media/4top.png)

按照接线图接好线，利用micro USB数据线上电，同时还需要外接电源 (4个AAA电池安装到电池盒，且保证电源充足)。

使用在线浏览器下载示例代码，则需要将下载好的 “.hex” 文件发送到micro:bit V2主板上。如下两种情况：

**① 下载示例代码(WebUSB功能)**

使用 **Google Chrome** 或  **Microsoft Edge浏览器**，需要先进行设备配对，再将示例代码下载到micro:bit V2主板上。

![Img](./media/Animation-08.gif)

**② 下载示例代码(非WebUSB功能)**

 如果是使用其他浏览器（如：非Google Chrome 或 非Microsoft Edge浏览器），将示例代码下载、发送到micro:bit V2主板上。

![Img](./media/Animation-008.gif)

将示例代码成功下载、发送到micro:bit V2主板之后，然后将micro USB数据线从micro:bit V2主板上拔下来。

将舵机扩展板上的左侧拨码开关拨到RGB端，右侧拨码开关拨到ON端。

![Img](./media/CBA01.png)

按下micro:bit V2主板上面的按钮A且松开时，翅膀慢速飞舞起来，同时SK6812 RGB LED亮红色灯，micro:bit V2主板上面的5×5 LED点阵屏显示图案![Img](./media/ab3.png)；

按下micro:bit V2主板上面的按钮B时，翅膀快速飞舞起来，同时SK6812 RGB LED亮绿色灯，micro:bit V2主板上面的5×5 LED点阵屏显示图案![Img](./media/ab6.png)；

同时按下micro:bit V2主板上面的按钮A+B时，翅膀停止飞舞，同时SK6812 RGB LED熄灭，micro:bit V2主板上面的5×5 LED点阵屏清屏。

![Img](./media/dongtu-008.gif)
